# vs-trans README

我太懒了，看代码的时候遇到不认识的单词，懒得打开浏览器，所以就在VSCode中搞了这么个插件。
之前的wordcount插件里有这个功能，但是百度接口菜🐔挂了，所以转用有道云的翻译接口。

额，有道云接口访问频次有限制，所以也不靠谱。因此曲线救国用了青云客的智能机器人作为翻译中转下。
